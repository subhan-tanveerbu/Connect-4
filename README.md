# 🎮 Connect 4 AI

An AI-powered Connect 4 game built using **Python**, **Pygame**, and **NumPy**, featuring an intelligent opponent driven by the **Minimax Algorithm**. The project demonstrates game development principles, board-state management, and AI-based decision-making through strategic gameplay.

---

## 📌 Overview

Connect 4 is a classic two-player strategy game where players take turns dropping pieces into a 7×6 grid. The objective is to connect four pieces horizontally, vertically, or diagonally before the opponent.

This implementation includes a computer-controlled AI that evaluates possible moves using the **Minimax Algorithm**, making gameplay both challenging and engaging.

---

## ✨ Features

- 🎮 Interactive graphical user interface with Pygame
- 🤖 AI opponent using the Minimax Algorithm
- 📊 Efficient board representation using NumPy
- 🏆 Automatic win detection
- 🔄 Turn-based gameplay mechanics
- 🎯 Horizontal, vertical, and diagonal win checking
- ⚡ Fast and responsive gameplay experience

## 🧠 AI Implementation

The game utilizes the **Minimax Algorithm** to evaluate possible future board states and determine the optimal move for the AI player.

### AI Features
- Move Evaluation
- Board Scoring
- Win Prediction
- Defensive & Offensive Strategies
- Recursive Game Tree Search
